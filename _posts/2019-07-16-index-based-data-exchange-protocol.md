This document describes how users can exchange data object in a decentralized, secure way using an indexer

-     Alice encrypts data object with a key for Alice and Bob
-     Alice publishes location of data object on the indexer
-     Bob fetches locations from indexer
-     Bob retrieves data object and decrypts it
