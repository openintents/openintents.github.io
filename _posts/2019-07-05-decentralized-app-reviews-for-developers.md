## OI App Center

OI App Center is a user interface for app reviews. It is build in a way that any website, app or location can be reviewed, however, for now the focus is on Blockstack app.

### How to add a Rate App link?

Developers can now add a link for app review to the decentralized review platform build on blockstack. A typical link looks like this:

*<a
  href="https://app-center.openintents.org/appco/1092/review"
  rel="noopener noreferrer">*

*  Rate App!*

*</a>*

The actually link is provided on [https://app-center.openintents.org](https://app-center.openintents.org) if you claim your app.

### How are reviews aggregated?

Users can store and publish (via Radiks) as many reviews as they want. However, only the latest review is displayed and considered when calculating the average score on OI App Center.
